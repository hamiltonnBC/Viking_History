# Viking Atlas

An interactive historical map that lets you travel through the Viking Age (c. 750–1408 AD). Watch raids, settlements, trade routes, and explorations unfold across the Norse world as you scrub through time.

Built with React, TypeScript, D3, and Vite.

---

## Repository Layout

```
/
├── viking-atlas-v2/   # Current application (React + TypeScript + Vite)
└── legacy/            # Original Vanilla JS + D3 prototype (archived)
```

The active project lives entirely inside `viking-atlas-v2/`. Everything below refers to that directory.

---

## Quick Start

```bash
cd viking-atlas-v2
npm install
npm run dev
```

The app runs at `http://localhost:5173` by default.
